# healthmetrics

## Overview

`healthmetrics` is an R package designed to help users easily check and categorize their health metrics, such as blood pressure, cholesterol levels, and body mass index (BMI).

## Installation

You can install the latest version of `healthmetrics` from GitHub using the `devtools` package:

```r
devtools::install_github("Obehiik/Obehiik.github.io")
```

Once installed, load the package into your R session to access its functions:

```r
library(healthmetrics)
```

# Functions

## **Blood Pressure Category**

To determine your blood pressure category, use the `BloodPressureCategory()` function:

```r
BloodPressureCategory()
```

## **BMI Calculation and Categorization**

Calculate and categorize your BMI using the `computeBMI()` function:

```r
computeBMI()
```

## **Cholesterol Levels Check**

Check your cholesterol levels with the `cholesterol_levels()` function:

```r
cholesterol_levels()
```
